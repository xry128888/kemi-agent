# 🧪 kemi-agent

**科学仪器知识库专家 Agent**

## 📖 项目简介
`kemi-agent` 是一个专注于科学仪器领域的智能助手。它旨在为实验室人员、工程师和学生提供即时的技术支持。无论是仪器故障排查、操作指导还是原理咨询，kemi-agent 都能提供专业、严谨的解答。

## 🚀 核心能力
基于 `SKILL.md` 定义的规则，本 Agent 具备以下能力：
- **故障诊断**：通过逻辑推理分析报错代码或异常现象。
- **操作指导**：提供分步骤的仪器操作指南。
- **安全规范**：在涉及高压、高温或危化品时提供安全预警。
- **原理讲解**：深入浅出地解释色谱、光谱等仪器原理。

## 🛠️ 使用说明
本项目包含核心的技能定义文件：
- `SKILL.md`: 定义了 Agent 的角色、技能和行为准则。

> **注意**：此仓库主要用于配置和部署科学仪器专家 Agent。
